# Installation

Follow these steps to install the Apex Legends configuration files.

## Step 1: User Folder

Replace `{name}` with your Windows username.

C:\Users\{name}\Saved Games\Respawn\Apex\local

## Step 2: Copy Settings Files

Copy the following files:

- settings.cfg
- videoconfig.txt

Paste them into:

C:\Users\{name}\Saved Games\Respawn\Apex\local

Launch Apex Legends once if the folder does not exist.

## Step 3: Install autoexec.cfg

Copy:

- autoexec.cfg

Paste it into the Apex Legends `cfg` folder.

Steam (default):
C:\Program Files (x86)\Steam\steamapps\common\Apex Legends\cfg

Origin (default):
C:\Program Files (x86)\Origin Games\Apex\cfg

Create the `cfg` folder if needed.

## Step 4: Enable Autoexec

Add this launch option:

+exec autoexec.cfg

Steam:
Right-click Apex Legends → Properties → Launch Options

## Optional: Read-Only Protection

To prevent overwrites:
- Right-click `settings.cfg` and `videoconfig.txt`
- Properties → Enable **Read-only**

## Done

Start Apex Legends and the configs should be active.
